## Pemilik Repo

## 🎮 Tentang
Ini adalah repositori untuk **APK Card Generator**, alat untuk membuat HTML otomatis untuk postingan MOD APK dengan desain dark UI dan animasi spinner.

## 🛠️ Tech Stack

- CSS3
- JavaScript
- Firebase (untuk komentar di Blogger)
- Python

- **MOD Request Form**
- **Firebase Comment System**
